## 🚀 核心项目（摘自仓库概览）

- **API 安全扫描器** — 半成品：支持未授权检测与 API 字典自动采集（仓库：vuln-scanner）  
- **AI 驱动渗透测试框架** — LLM 与渗透流程对接，自动生成攻击向量与验证步骤（仓库：ai-penetration-testing）  
- **APP 安全扫描器** — 已完成的移动端自动化检测工具  
- **业务风险梳理脚本** — 通过模拟风控对抗批量识别业务流程薄弱点  
- **LLM 安全模型实验** — 基于 8B/32B 模型的安全推理与辅助决策组件
